## css 정렬

#### margin 중앙 정렬
> margin: auto 

#### table 정렬
> display: table;  
> 
> display: table-cell;  
> vertical-align: middle;  
 
#### flex 중간 가로 정렬
> display: flex;  
> justify-content: center;  
> align-items: center;  
>   
> display: flex;   
> flex-direction: row;  
> align-items: center;  
 
#### flex 중간 세로 정렬
> display : flex;  
> justify-content: center;  
> align-items: center;  
> flex-direction: column;  

#### absolute 부모요소 기준 가운데
> position: absolute;  
> top: 50%;  
> left: 50%;  
> transform: translate(-50%, -50%);  

#### 글 중간 정렬
> text-align: center;  
> line-height: 영역높이만큼  

#### img 이미지 가운데 정렬

> margin-left: auto;
